## BookStore

* Código construído baseado no curso https://app.balta.io/courses/1939. Foi adaptado de **.net framework** para **.net core**.
* A aplicação está conteinerizada, ou seja, a aplicação em um conteiner e o banco de dados em outro. Para testá-la é necessário o arquivo **docker-compose.yml**.
* Execute o comando abaixo no diretório onde o arquivo *docker-compose.yml* foi baixado.
    ```
    docker-compose up
    ```
